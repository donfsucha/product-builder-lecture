# 가위바위보 & 제휴 문의 프로젝트

가위바위보 게임과 비즈니스 제휴를 위한 문의 폼이 결합된 간단한 웹 애플리케이션입니다.

## 주요 기능
- **가위바위보 게임**: 사용자가 가위, 바위, 보 중 하나를 선택하면 컴퓨터와 승부하고 결과를 상세한 설명과 함께 보여줍니다.
- **제휴 문의 폼**: Formspree 서비스를 이용하여 사용자의 제휴 문의를 이메일로 직접 전달받을 수 있습니다.

## 기술 스택
- HTML5
- CSS3 (Apple/Modern Style)
- JavaScript (Vanilla JS)
- [Formspree](https://formspree.io/) (백엔드 없는 폼 처리)
- [OpenMoji](https://openmoji.org/) (아이콘 이미지)
